Sì, Telnyx offre diversi strumenti per testare le API:
Postman Collections
Puoi utilizzare le nostre collezioni Postman per testare le API senza scrivere codice. Disponibili su postman.com/telnyx-api con esempi per configurare l'ambiente locale e inviare richieste di test.
Mock Brands e Campaigns (10DLC)
Per testare il flusso 10DLC, puoi creare mock brands e campaigns gratuitamente tramite API o Mission Control Portal, impostando il campo "mock" su true.
WebRTC Test Application
L'app webrtc.telnyx.com ti permette di testare chiamate usando credenziali SIP o JWT token per verificare la configurazione.
Web Dialer
Strumento nel Mission Control Portal per fare chiamate di test senza PBX, utile per debug di problemi vocali.
Test Messaging
Puoi acquistare due numeri Telnyx per testare messaggi on-net immediatamente senza requisiti di registrazione.
Tutti questi strumenti sono accessibili dal tuo account Mission Control Portal.

intendo test per l'acquisto dei numeri
• 8m
Per testare l'acquisto di numeri, ti consiglio di iniziare con numeri US o CA che non hanno requisiti normativi per l'attivazione immediata.
Test con numeri US/CA:
Cerca numeri disponibili: GET /v2/available_phone_numbers?filter[country_code]=US
Ordina il numero: POST /v2/number_orders con phone_number e connection_id
L'ordine sarà pending inizialmente, poi controlla lo stato con polling
Test messaging:
Puoi acquistare due numeri Telnyx per testare messaggi on-net immediatamente senza requisiti di registrazione. Vai su Numbers > Search & Buy nel portal, cerca numeri con feature SMS abilitata, e acquistane due per testare sia invio che ricezione.
Postman Collections:
Usa le nostre collezioni Postman per testare le API senza scrivere codice. Ogni pagina di riferimento API ha un pulsante "Run in Postman" per importare e testare immediatamente.
Inizia con numeri US/CA per evitare complicazioni normative durante i test.
